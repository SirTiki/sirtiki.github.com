Introducing Tiki!

The first ever JavaScript browser package manger.

  • Cross-domain module-level caching
  • 1 HTTP request
  • Automatic dependency resolution and versioning
  • Hosted: no installation or configuration

For a while now, JavaScript script loading and module loading has been heating up, with many loaders competing to address the painful issue of delivery your JavaScript in a performant manner. Tiki solves all that by leveraging several clever localStorage techniques.
